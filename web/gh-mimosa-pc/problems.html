  • Q：登录密码设置的要求？

    A：密码建议包含数字和英文字母（区分大小写），至少6位字符，不超过16位字符。

  • Q：交易密码设置的要求？

    A：需要6位纯数字密码，投资赎回时需要输入交易密码才可以进行。

  • Q：登录密码和交易密码有什么区别？

    A：登录密码是用户登录账户时使用的密码，交易密码是用户进行投资、赎回等涉及资金操作时使用的密码。

  • Q：如何更换已注册的手机号码？

    A：为了用户的账号资金安全，若用户的注册手机号注销、挂失或更换。请致电客服热线，由客服人员为您提供更换注册手机号码的相关服务。您需要提供以下材料：
      1. 本人身份证正反面照片
      2. 本人手持身份证正面照片（身份证上的内容清晰）
      3. 免责声明
      4. 请将以上资料发送至客服邮箱：service@2hlc.com

  • Q：用户账号或密码错误怎么办？

    A：同一账号连续输错5次密码，系统锁定该账号一天，一天后自动解锁，若已连续输错2次，建议使用密码找回功能。

  • Q：对获取动态密码的手机号码有什么要求？

    A：获取动态密码的手机号必须是您在二货理财平台注册的手机号码，限大陆地区电信运营商支持的号段且不支持小灵通，以及香港、台湾地区手机号码。

  • Q：登录失败的原因？

    A：请核对以下几点，这些可能是导致登录失败的原因：
      1. 确认输入的用户名是否正确。
      2. 确认输入的密码是否正确。如果忘记密码，可以点击”忘记密码“重新设置。
      3. 网络问题也会造成登录失败，如果提示网页有错误，建议稍后尝试。

  • Q：姓名中有符号或者是英文是否可以认证？

    A：抱歉，平台暂不支持。

  • Q：使用护照号、军官证号是否可以认证？

    A：为确保投资人资金安全，目前仅支持大陆居民二代身份证证件号进行认证。

  • Q：我认证成功后，是否可以更改证件号码？

    A：抱歉，平台暂不支持更改。

  • Q：为什么实名认证失败？

    A：
      1. 核对姓名、身份证号是否正确。
      2. 只支持中国大陆居民二代身份证，暂不支持其他认证方式。
      3. 是否更改过姓名、身份证号信息等。
      4. 户口是否做过迁移，户籍在落实之前会有一段冻结期。

  • Q：银行卡绑定失败的原因？

    A：
      1. 银行卡号输入有误，请重新核对卡号输入。
      2. 银行超时或银行系统异常，建议关闭网页后重新操作。
      3. 输入姓名有误，可能绑定的不是实名认证一致的银行卡，建议换与实名认证一致的银行卡尝试。
      4. 银行预留手机号有误，可能的原因有：
        A. 银行柜台提供的预留手机号和绑卡时输入的手机号不一致会导致鉴权失败。
        B. 银行预留的手机号只保存在用户的个人基本信息记录里面，还需要用户到柜台申请该手机号作为银行预留手机号用作快捷支付才能使用。
        C. 注意：能接收验证码的手机号不一定是柜台预留手机号，建议用户持身份证和银行卡去银行办理修改或者更换其他的银行卡绑定。

  • Q：如何更换、解绑认证的银行卡？

    A：为了保证投资人资金安全，投资与赎回功能实行同卡同出规则。若用户的银行卡注销掉、已挂失或是补换卡需要更换银行卡。请致电客服热线，由客服人员为您提供更换银行卡的相关服务。您需要提供以下材料：
      1. 注册手机号（用户本人注册手机号致电客服热线）。
      2. 原卡及新补换卡号。
      3. 本人手持身份证正面照（照片里身份证信息清晰）。
      4. 有效的银行卡挂失、注销、升级换卡证明，或银行近三个月流水（清晰可辨，无ps）。
      5. 免责声明。
      6. 请将以上资料发送至客服邮箱：service@2hlc.com。

  • Q：我可以绑定别人的银行卡吗？

    A：为保证投资人的利益，投资人只能绑定与本人身份信息一致的银行卡，不能绑定他人的银行卡。

  • Q：如何查询充值提现记录？

    在登录状态下，选择“我的”菜单，点击“提充记录”可以查询。

  • Q：充值成功后会有短信通知吗？

    A：会短信通知。

  • Q：申请提现，什么时候可以到账？

    A：当天提现，下一工作日到账，节假日申请提现最迟于T+2个工作日到账。具体到帐时间与绑卡银行处理速度有关。

  • Q：我可以把钱提现到别的卡上吗？

    为保证投资人的利益，投资人只能绑定与本人身份信息一致的银行卡，不能把钱提现到他人的银行卡。

  • Q：充值未到账该如何处理？

    A：
      1. 您输入的验证码或交易密码错误：请输入正确的验证码或交易密码。
      2. 银行卡余额不足：请您更换银行卡充值。
      3. 充值超出限额：请联系银行查询及更改支付额度。
      4. 您的银行卡未开通网上银行：请前往银行柜台或通过银行官网进行开通。
      5. 您的银行卡未开通银联在线支付：请您开通银联在线支付业务后再尝试充值。
      6. 页面提示超时或网络不稳定：请您切网络或者稍后再重试。
      7. 验证码接收不到：请您检查手机短信屏蔽设置，或联系客服核实验证码发送情况。
      8. 银行数据传输异常，导致订单超时等情况：需要刷新页面重新操作。
      9. 银行卡已过期、作废、挂失：请更换银行卡或将绑定银行状态恢复正常。
      10. 没有跳转到充值页面：请您检查是否浏览器拦截了弹出的页面，或者更换浏览器尝试。
      11. 如您有遇到其他问题或以上方法未能解决问题，请联系人工客服。
